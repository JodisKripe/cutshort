What is Cutshort?

Cutshort.in is a URL Shortner built by Rahul Tandon with love in India.

If you have very long URLs for example https://www.amazon.in/btgalitz-Rick-Roll/dp/B07K6QP8RF/ can be changed to simply http://www.cutshort.in/wwrno
